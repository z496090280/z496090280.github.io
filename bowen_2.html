highlight.js的应用于实践

  • 分类：插件篇
  • 发表于：2017-07-04

先来看看 highlight.js 有什么能力吧：

  • 支持 71 种编程语言的语法解析；拥有 44 种样式
  • 自动检测编程语言
  • 同时为多种编程语言代码高亮
  • 可以在 node.js 平台上运行
  • 支持各种标签
  • 与任何 js 框架兼容

OK，接下来就看看怎么用 hightlight.js 吧！

引入 highlight.js 文件

        
            <link href="http://cdn.bootcss.com/highlight.js/8.0/styles/monokai_sublime.min.css" rel="stylesheet">
            <script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/highlight.min.js"></script>
            <script >hljs.initHighlightingOnLoad();</script>

1.第一个文件是样式文件，我们采用 sublime 编辑器类似的样式

注意: 这里采用Bootstrap中文网 提供的 CDN 链接，而非官网提供的 CDN（本人用的时候官网版本9.12.0版本，貌似有一些不明的问题）

2.第二个文件是 highlight.js 的核心文件，就是它解析代码的。

3.第三个文件是调用 hihglight.js 解析代码,把这三步全部放在head标签里面！

body部分准备工作

      
          <pre><code class="css">...</code></pre>

所有代码写在code的包裹中，其中class指定语言类型，案例中已css为例。

css代码高亮实例：

        
            body{
                background:#f1f1f1;
            }
            .content{
                background:#fff;
            }
            blockquote{
                border-left:5px solid #0593d3;
            }
            h2{
                margin-bottom: 50px;
                text-align: center;
            }
            h3{
                font-weight: bold;
            }
            p{
                font-size: 18px;
                margin-bottom: 20px;
            }

javascript代码高亮实例：

        
            var docEl = document.documentElement;
            function getSize() {// 获取屏幕的宽度
                var screenWidth = docEl.clientWidth;
                docEl.style.fontSize =  screenWidth / (640/60)  + 'px';
            }
            getSize();// 页面加载完执行一次
            window.onresize = function() {
                getSize();
            }

html代码高亮实例：

        
            <ul class="dropdown-menu">
                <li><a href="#tab-Chrome">chrome浏览器</a></li>
                <li><a href="#tab-Firefox">Mozilla Firefox</a></li>
                <li><a href="#tab-Safari">Safari</a></li>
                <li role="separator" class="divider"></li>
                <li><a href="#tab-Opera">Opera浏览器</a></li>
                <li role="separator" class="divider"></li>
                <li><a href="#tab-IE">IE</a></li>
            </ul>
最坑的就是html应用了！！！

为什么说html最坑呢，因为你直接复制html文档进去他会解析成html文档流

解决方案：把所有的标签括号全部换成代码格式，就像这样：

<用&lt;表示
>用&gt;表示

番外篇:在页面输出&lt;需要代码转义&amp;lt;，不然在页面输出就是<了。。。

总结

highlight.js 支持 176 种编程语言（完整列表看这里），在你书写这些代码的时候，可以看看 highlight.js 是否支持所用的语法，然后标注对应的语言，让 highlight.js 完美的对其高亮显示吧。

本页面中的代码高亮就是上面的方式哦，直接开启浏览器的调试工具研究吧！喜欢分享一下哦..
